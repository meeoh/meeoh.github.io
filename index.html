Slowed And Reverbed Bot

An automated music manpiulator

Slowed and reverbed is a genre of music that takes music (usually rap or r&b) and simply applies slowing and reverb effects to the music. This bot works by interacting with reddits api every night at 9:00pm EST (on a cronjob). It checks r/hiphopheads for any new music that meets the following critera:

  1. Is 'fresh' in the title
  2. Is it a youtube submission
  3. Is it NOT a music video
  4. Is it NOT a cypher
  5. Is it NOT a playlist
  6. It has at least 100 upvotes on reddit

If all the criteria is met, the bot downloads the song, applies slowing and reverbing affects using python, and reuploads the song to youtube. It then responds to the original reddit thread with a comment linking to the slowed and reverbed version as well as posting on my own subreddit r/slowedandreverbed.

Energy Usage Reporter

An automated energy usage reporter

After moving into a place that uses a company called Provident Enery for hydro, I decided to create a energy reporter. The idea of the reporter is that every day it reports a number that represents the percentage change in energy from the day before yesterday to yesterday. That way, you are able to monitor electricity usage on a daily basis and catch overages before they happen (e.g leaving the laundry on). It simply uses python to scrape the provident reporting site, and then parses the percentage change and sends it over telegram. This was designed with an adapter pattern, so it is very easy to create a new messaging adapter, and add that into the code if you wanted messages via a different communication platform.
